# SQL To-Do List Management System

A command-line To-Do List application built using Python and SQLite.

## Features

- Add Tasks
- View Tasks
- Update Tasks
- Delete Tasks
- Mark Tasks as Completed
- Search Tasks
- Filter Pending/Completed Tasks

## Technologies

- Python 3
- SQLite
- SQL

## Database

SQLite stores all task records inside:

todo.db

## Run

Install Python.

Clone repository.

Run

python main.py

```
